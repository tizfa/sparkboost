Class MpBoostLearner

  • java.lang.Object
    • it.tizianofagni.sparkboost.MpBoostLearner

  • public class MpBoostLearner
extends Object
    A Spark implementation of MP-Boost algorithm, an improved variant of the well known AdaBoost.MH boosting algorithm.

    The original article describing the algorithm can be found at http://link.springer.com/chapter/10.1007/11880561_1.
    Author:
    Tiziano Fagni (tiziano.fagni@isti.cnr.it)

        public BoostClassifier buildModel(String libSvmFile,
                                  boolean labels0Based,
                                  boolean binaryProblem)
        Build a new classifier by analyzing the training data available in the specified input file. The file must be in LibSvm data format.
        Parameters:
        libSvmFile - The input file containing the documents used as training data.
        labels0Based - True if the label indexes specified in the input file are 0-based (i.e. the first label ID is 0), false if they are 1-based (i.e. the first label ID is 1).
        binaryProblem - True if the input file contains data for a binary problem, false if the input file contains data for a multiclass multilabel problem.
        Returns:
        A new MP-Boost classifier.
